¡Latiñol!

Latiñol es una nueva propuesta de escritura para el español

Español

El español tiene muchas letras que se pronuncian igual y palabras que se pueden escribir de muchas maneras. El latiñol busca arreglar esto con una nueva propuesta: Eliminar aquellas letras que se pronuncian igual que otras letras.
